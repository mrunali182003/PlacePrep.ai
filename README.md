# PlacePrep.ai 🚀

AI-powered placement preparation platform for students.

## Features
- 📄 AI Resume Analyzer — score, skills, suggestions
- 🎤 Mock Interview — AI interviewer with feedback
- 🎙️ Voice Interview — speak your answers
- 💻 Coding Practice — live code editor
- 📄 Resume Builder — generate PDF resume
- 💼 Job Recommendations — based on your skills
- 📊 Progress Tracking — charts and analytics
- 🏆 Leaderboard — compete with peers

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB
- **AI:** Python, Flask, Google Gemini API
- **Auth:** JWT

## Setup
```bash
# Backend
cd backend && npm install && node server.js

# AI Service  
cd ai-service && python resume_ai.py

# Frontend
cd frontend && npm install && npm run dev
```
